## Español

### 📋 Descripción del Proyecto

**Homely** es una plataforma backend para la gestión y visualización de inmuebles funcionando de manera similar a
aplicaciones populares como **Fotocasa** o **Idealista**.

Este proyecto proporciona una API REST robusta que permite:

- 🏘️ Listar, crear, actualizar y eliminar propiedades inmobiliarias
- 🔍 Búsqueda y filtrado avanzado de inmuebles
- 💰 Gestión de precios y ofertas
- 👤 Autenticación y autorización de usuarios
- 📸 Gestión de imágenes de propiedades
- ⭐ Sistema de valoraciones y comentarios
- 📍 Búsqueda por ubicación geográfica

### 🛠️ Tecnologías Utilizadas

- **Java 25** - Lenguaje de programación
- **Spring Boot 4.1.0** - Framework web y base de datos
- **Maven** - Gestor de dependencias y construcción
- **Maven Wrapper** - Para asegurar compatibilidad de versiones

### 📋 Requisitos Previos

- Java JDK 25 o superior
- Maven 3.8+
- Git
- PostgreSQL

### 🚀 Instalación y Ejecución

#### **1. Clonar el repositorio**

```bash
git clone https://github.com/Luis-Alberto-Lara-Martinez/homely-backend.git
```

#### **2. Compilar el proyecto**

```bash
./mvnw clean install
```

#### **3. Ejecutar la aplicación**

```bash
./mvnw spring-boot:run
```

En Windows:

```cmd
mvnw.cmd spring-boot:run
```

#### **4. Acceder a la API**

- La aplicación estará disponible en: `http://localhost:8080`

## English

### 📋 Project Description

**Homely** is a backend platform for managing and displaying real estate properties, functioning in a similar way to
popular applications like **Fotocasa** or **Idealista**.

This project provides a robust REST API that enables:

- 🏘️ List, create, update, and delete real estate properties
- 🔍 Advanced search and filtering of properties
- 💰 Price and offer management
- 👤 User authentication and authorization
- 📸 Property image management
- ⭐ Rating and comment system
- 📍 Geographic location search

### 🛠️ Technologies Used

- **Java 25** - Programming language
- **Spring Boot 4.1.0** - Web framework and data access
- **Maven** - Dependency management and build tool
- **Maven Wrapper** - Ensures version compatibility

### 📋 Prerequisites

- Java JDK 25 or higher
- Maven 3.8+
- Git
- PostgreSQL

### 🚀 Installation and Execution

#### **1. Clone the repository**

```bash
git clone https://github.com/Luis-Alberto-Lara-Martinez/homely-backend.git
```

#### **2. Build the project**

```bash
./mvnw clean install
```

#### **3. Run the application**

```bash
./mvnw spring-boot:run
```

On Windows:

```cmd
mvnw.cmd spring-boot:run
```

#### **4. Access the API**

- The application will be available at: `http://localhost:8080`
